# Playwright E2E Test Example - POM + Actions

Este proyecto es un ejemplo de pruebas E2E usando **Playwright**, con:

- **Page Object Model (POM)**: cada página define locators y métodos básicos de UI.
- **Actions**: clases que representan flujos de usuario completos sobre la UI.
- **Validaciones**: separadas en Page o Actions según corresponda.
- **Test Data / Factories**: datos centralizados para tests.
